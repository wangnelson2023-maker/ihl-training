---

## 第一步：打开 CloudStudio 网站

1. 打开你的浏览器（推荐用 Chrome 浏览器）
2. 在地址栏输入：**https://cloudstudio.net**
3. 按下回车键，进入 CloudStudio 首页
4. 点击页面右上角的 **「登录」** 按钮
5. 用你的 **微信扫码** 或 **手机号** 登录

> 📌 什么是 CloudStudio？
> CloudStudio 是腾讯的一个「在线编程环境」。你可以把它理解成一台在云端运行的电脑——你不需要自己买服务器，它帮你运行你的网站，并给你一个可以公开访问的网址。

---

## 第二步：创建工作空间

1. 登录成功后，点击页面上的 **「新建工作空间」** 按钮（通常在页面中间或左上角）
2. 会出现一个弹窗，选择 **「空白模板」** 或 **「Node.js」** 模板
3. 给工作空间起个名字：**伊赫莱培训**（随便叫什么都可以）
4. 点击 **「创建」**，等待几秒钟，工作空间就创建好了

> 创建完成后，你会看到一个类似 VS Code 的编辑器界面——这就是你的「云端电脑」。

---

## 第三步：上传项目文件

现在你需要把电脑上的项目文件上传到 CloudStudio。

### 方法 1：直接拖拽上传（推荐，最简单）

1. 在你的电脑上，打开文件夹：`C:\Users\Nelson\WorkBuddy\2026-05-28-16-39-43\deploy`
2. 你会看到这些文件：
   ```
   📁 deploy/
   ├── 📄 server.js          ← 后端服务器
   ├── 📄 package.json       ← 项目配置（依赖声明）
   ├── 📄 index.html         ← 主H5训练页面
   ├── 📄 dashboard.html     ← 管理者看板
   └── 📄 qr.html            ← 扫码入口页
   ```
3. 在 CloudStudio 左侧，你会看到一个文件列表区域
4. 选中上面这 **5个文件**，用鼠标拖到 CloudStudio 的文件列表区域
5. 松开鼠标，文件就会上传

### 方法 2：如果 CloudStudio 不支持拖拽

CloudStudio 左侧有个 **上传按钮**（通常是 ↑ 图标），点击后选择文件上传。

> ⚠️ 确保上传的是 **deploy 文件夹里面的文件**，而不是 deploy 文件夹本身！

---

## 第四步：安装项目依赖

上传完文件后，你需要安装项目运行所需的「依赖包」。

1. 在 CloudStudio 底部，找到 **「终端」(Terminal)** 面板
   - 如果看不到终端，点击顶部菜单 **「终端」→「新建终端」**

2. 在终端里，你会看到一个闪烁的光标，输入以下命令然后按回车：

   ```
   npm install
   ```

3. 等待约 10-30 秒，你会看到一些文字滚动。最后出现类似 `added 60 packages` 就说明安装成功了。

   > 如果出现红色的报错信息，先输入 `node -v` 检查 Node.js 是否安装。CloudStudio 的 Node.js 模板默认已安装。

---

## 第五步：启动服务器

依赖安装完成后，就可以启动你的网站了！

1. 在终端输入以下命令然后按回车：

   ```
   node server.js
   ```

2. 如果看到以下输出，说明启动成功：

   ```
   伊赫莱训练平台已启动: http://localhost:3000
   前端页面: http://localhost:3000
   管理者看板: http://localhost:3000/dashboard.html
   ```

3. **⚠️ 重要：不要关闭这个终端！** 关闭终端 = 网站停止运行。让它一直开着就好。

---

## 第六步：获取公网访问地址

网站已经在云端跑起来了，但 `localhost:3000` 是内部地址，别人访问不到。你需要获取一个公网 URL。

### CloudStudio 获取公网地址的方法：

1. 看 CloudStudio 界面的右下角或底部状态栏，找一个叫 **「端口」** 或 **「Ports」** 的标签
2. 你应该能看到一个 `3000` 的端口
3. 在 3000 端口那一行，应该有一个 **🌐 小地球图标** 或者 **「公开」/「Share」** 之类的按钮
4. 点击它，CloudStudio 会生成一个公网地址，类似：
   ```
   https://xxxxx-3000.cloudstudio.com
   ```
5. **复制这个地址**，这就是你的 H5 训练网站的公开地址！

> 不同版本的 CloudStudio 界面可能略有不同：
> - 有些版本是点击端口号前面的 🔗 图标
> - 有些版本是右键端口号 → 选择「设为公开」
> - 有些版本需要点击「生成访问链接」

## 第八步：保持网站一直在线（重要！）

CloudStudio 的工作空间如果长时间不活动，可能会自动休眠。

### 如何保持在线：

1. **方法一**：如果 CloudStudio 支持「常开模式」，打开它
2. **方法二**：每隔几小时访问一下页面，保持活跃
3. **方法三**：如果 CloudStudio 有「保活」功能，在设置中开启

> 💡 如果发现网站打不开了，回到 CloudStudio，重新在终端运行 `node server.js` 即可。

## 附：更新代码的方法

如果你以后修改了代码，需要同步到 CloudStudio：

1. 打开 CloudStudio 工作空间
2. 在终端按 **Ctrl + C** 停止服务器
3. 删除旧文件，拖入新文件覆盖
4. 重新运行 `node server.js`

## 附：常见问题

### Q: 上传文件后 npm install 报错？
A: 确认 `package.json` 已上传，然后重新运行 `npm install`

### Q: 公网链接打不开？
A: 
1. 确认终端里 `node server.js` 是否还在运行
2. 确认端口 3000 是否已设为「公开」
3. 尝试在普通浏览器中打开（不用微信），看是否是微信的限制

### Q: AI 对话功能不能用？
A: 
1. 点击页面右上角 ⚙️ 按钮
2. 检查 AI Key 是否已配置
3. 确保你的 DeepSeek API Key 有余额

### Q: 数据存储在哪里？
A: 所有练习记录存储在 CloudStudio 工作空间的 `data/records.json` 文件中。注意：如果工作空间被删除，数据也会丢失。
